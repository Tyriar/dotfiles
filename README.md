# Daniel Imms' dotfiles

My dotfiles for Linux and Windows.

![](terminal.png)

## Dependencies

Node.js is required to run this project.

## Installing/syncing environment

```bash
npm install
npm start install
```

## Installing a single module

```bash
npm start install <module>
```
